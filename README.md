# Setup for React.js

```bash

# Abra o terminal/CMD e clone este repositório
$ git clone https://github.com/victorshgo/setup-react.git

# Acesse a pasta do projeto
$ cd setup-react

# Instale as dependências
$ yarn

# Execute a aplicação
$ yarn start

```
